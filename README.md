# GET LOCATION WEATHER

## Running

``$ git clone https://github.com/gabrielmaxgb/get-location-weather.git ``  
> Navigation to the source location and then:   

``$ yarn``   
``$ yarn start``   

> Don't forget to allow browser geolocation =D     

## Technologies:

- ReactJS
- Material UI
- Mui Icons
- Motion
- Open Weather API
