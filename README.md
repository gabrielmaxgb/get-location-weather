# Running

``$ git clone https://github.com/gabrielmaxgb/get-location-weather.git ``  
``$ yarn``   
``$ yarn start``   

> Don't forget to allow location =D     

## Technologies:

- ReactJS
- Material UI
- Mui Icons
- Motion
- Open Weather API
